# FOCUSLINE

RSVP Nano의 읽기 방식을 브라우저로 옮긴 백엔드 없는 정적 웹앱입니다. 설정과 작성 중인 글은 `localStorage`, 서재와 책별 진행률은 `IndexedDB`에 저장되며 데이터가 브라우저 밖으로 전송되지 않습니다.

## 실행

빌드 과정이나 패키지 설치가 없습니다.

```bash
cd rsvp-reader
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 엽니다. `index.html`을 직접 열어도 핵심 기능은 동작하지만 전체화면·화면 방향 잠금은 로컬 서버에서 더 안정적입니다.

## 기능

- 10–1000 WPM 조절
- RSVP Nano와 같은 ORP 위치 계산과 초점 글자 강조
- 아주 긴 단어는 화면 안에 맞게 자동 축소하면서 ORP를 정중앙에 유지
- 긴 단어·문장부호 추가 지연
- 이전/다음 주변 단어 표시
- 다크·세피아·라이트 테마
- 강조된 ORP 글자를 모든 기기의 화면 정중앙에 고정
- 글자 크기·자간·글꼴 조절
- 전체화면과 가로 방향 잠금 시도
- `읽기 시작` 뒤 첫 단어에서 대기하고 사용자가 재생할 때 진행
- 재생 중 조작 UI와 주변 단어를 페이드아웃하고, 화면을 건드리면 잠시 다시 표시
- 진행률, 남은 시간, 문장 처음으로 되감기
- 일시정지 후 화면 왼쪽 50%/오른쪽 50%를 짧게 눌러 1단어 이동, 길게 눌러 현재 WPM 간격으로 연속 탐색
- 책을 서재에 저장·수정·삭제하고 책마다 마지막 읽은 위치에서 이어 읽기
- `Space` 재생/정지, 방향키 이동·속도 조절, `R` 문장 되감기, `E` 편집 화면

## 구조

- `index.html`: 설정 화면과 몰입형 리더 마크업
- `styles.css`: 반응형 레이아웃과 세 테마
- `app.js`: 토큰화, ORP, 페이싱, 재생 상태, 로컬 저장
- `library.js`: IndexedDB 책 목록, 저장, 수정, 삭제, 책별 진행률

외부 폰트, 라이브러리, API를 사용하지 않습니다.
